# Cloud Foundations (Practitioner)

This repository contains my hands-on AWS tasting for learning and revision.

## Topics 
1. **Introduction to AWS IAM** – Learn about users, groups, roles, and permissions.
2. **Build VPC and Launch Web Server** – Set up a Virtual Private Cloud and deploy a web server.
3. **Introduction to Amazon EC2** – Launch and connect to EC2 instances.
4. **Working with EBS** – Attach and manage Elastic Block Store volumes.
5. **Build a Database Server** – Deploy and configure a database server on AWS.
6. **Scale & Load Balance your Architecture** – Implement auto-scaling and load balancing.
